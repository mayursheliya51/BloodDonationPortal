This portal can be used by doctors in different areas in order to get help from donners in an emegency sitation if he/she is willing to donate the blood.

How it works

  • Any person who wants to donate blood in emergency situations can register through this portal.
  • He/She must select user type as "donner" while registering. Ones registration is successful a doneer will get notification if there is an emergency in nearest location.
  • One more important side of this portal is doctor. A user can regiater As a doctor in this portal.
  • User who registered as a docyor will be able to see a form.He can enter the location of the emergency and select required blood group.
  • The portal will show all near by donners available to that location if exist.
  • After that doctor can enter the location and can notify all the donners by sending a quick sms to all. Along with This there is a super user in this portal who can manage all the donners and doctors who have registered through this portal. Super user can view, modify and delete any user's details.
